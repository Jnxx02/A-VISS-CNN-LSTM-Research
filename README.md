# A-VISS-CNN-LSTM-Research

Research repository for nurse stress classification using Subject-Dependent CNN-LSTM and multimodal physiological sensor data.

## Dataset
Nurse Stress Prediction Wearable Sensors Dataset (Kaggle)

## Workflow
- Data Cleaning
- Per-Subject Standardization
- Sliding Window Segmentation
- CNN-LSTM Training
- Model Evaluation

## Best Result
- Accuracy: 85%
- Weighted F1-Score: 86%
